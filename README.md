## Introduction

The Image Paginated App is a React.js application that allows users to browse through images with pagination. This app helps to load images efficiently by displaying a limited number of images per page, improving performance and user experience.

## Features

- Display images in a paginated format
- Navigate through pages
- Responsive design
- Lazy loading of images

## Installation

Follow these steps to get a local copy of the project up and running:

1. Clone the repository:
    ```sh
    git clone https://github.com/your-username/image-paginated-app.git
    ```
2. Navigate to the project directory:
    ```sh
    cd image-paginated-app
    ```
3. Install dependencies:
    ```sh
    npm install
    ```

## Usage

To start the development server, run:
```sh
npm start
```

This will run the app in the development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
